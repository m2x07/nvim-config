**Requirements:**

- [neovim](https://neovim.io/) >= v0.9.5
- [node-js](https://nodejs.org) (optional, but required for some parsers)
- ripgrep
- [a nerd font](https://nerdfonts.com/) (optional)

**Install/Update:**

```bash
git clone https://github.com/m2x07/nvim-config $HOME/.config/nvim
```
